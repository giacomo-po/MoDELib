Detailed Description

template<typename Derived, typename FieldType>
struct model::FieldPointBase< Derived, FieldType >

A class template that contains the underlying FieldType data of the Derived particle.

If MPI is not used, the data is stored locally in the base class Eigen::Matrix<scalar,rows,cols>.

In case that MPI is used, the underlying data is not stored with the object but is delocalized to a static vector that can efficiently be syncronized using MPI_Allgatherv. However, the base data is conveniently used as regular Eigen object via the Eigen::Map class.
