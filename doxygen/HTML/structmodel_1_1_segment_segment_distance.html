Detailed Description

template<int dim>
struct model::SegmentSegmentDistance< dim >

Class template which computes the distance between two lines segments in dimension dim. Algorithm adapted from V. LUMELSKY. ON FAST COMPUTATION OF DISTANCE BETWEEN LINE SEGMENTS. Information Processing Letters 21 (1985) 55-61.
