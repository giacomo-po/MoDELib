Detailed Description

template<char prefix, bool autoDelete = true>
class model::SequentialOutputFile< prefix, autoDelete >

A class template for outputing data to files with automatic sequential filename. Instances of SequentialOutputFile can be used analogously to std::cout. The first template parameter is the prefix of the filename. The second template parameter defines the behaviour of the class when an existing file with the same name is found.

Example: create 4 files named A_0, A_1, A_2, A_3 containing int 0,1,2,3 respectively.

for (int k=0;k<4;++k)
{
model::SequentialOutputFile<'A',1> file;
file<<k;
}
